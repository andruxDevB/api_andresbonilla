Prerequisitos:
- Java 17 (como variable de entorno)
- Gradle 7.6.1 (como variable de entorno)
- Karate 1.4.1 (como variable de entorno)

Ejecución:
1. Clonar el repositorio
    - git clone https://github.com/andruxDevB/api_andresbonilla.git
2. Ejecutar
    - ./gradlew clean test (desde consola en la raíz del proyecto)
3. Abrir 
    - build/karate-reports/karate-summary.html (en el navegador) para ver el reporte de pruebas
